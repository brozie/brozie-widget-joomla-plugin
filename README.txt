Brozie Widget Joomla Plugin
==========================

The [Brozie Widget Joomla Plugin](http://www.freechatforwebsite.com/) seamlessly 
integrates Brozie into your Joomla powered Website.  This combines the robust publishing capabilities
of Joomla with the power of Brozie.

This plugin is for Joomla 1.5 or greater (tested to 2.5).
